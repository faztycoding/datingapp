# ร้านกัญชา Premium Cannabis

เว็บไซต์ร้านกัญชาคุณภาพสูง สร้างด้วย React.js และ TailwindCSS

## คุณสมบัติ

- 🌿 แสดงสินค้ากัญชาหลากหลายสายพันธุ์
- 🔍 ระบบค้นหาและกรองสินค้า
- 📱 รองรับการแสดงผลบนมือถือ
- 🎨 ดีไซน์ที่ทันสมัยและสวยงาม
- ⭐ ระบบให้คะแนนและรีวิว
- 🛒 ตะกร้าสินค้า (พร้อม UI)

## เริ่มต้นใช้งาน

### ติดตั้ง Dependencies

```bash
npm install
```

### รันโปรเจค

```bash
npm start
```

เปิดเบราว์เซอร์ที่ [http://localhost:3000](http://localhost:3000)

## โครงสร้างโปรเจค

```
src/
├── components/
│   ├── Header.jsx          # ส่วนหัวของเว็บไซต์
│   ├── Hero.jsx            # หน้าแรก
│   ├── Products.jsx        # หน้าสินค้า
│   ├── ProductCard.jsx     # การ์ดสินค้า
│   └── Footer.jsx          # ส่วนท้ายของเว็บไซต์
├── data/
│   └── products.js         # ข้อมูลสินค้า (seed data)
├── App.jsx                 # คอมโพเนนต์หลัก
├── index.css               # สไตล์หลัก
└── index.js                # จุดเริ่มต้นของแอป
```

## สายพันธุ์ที่มีให้

- **อินดิก้า (Indica)**: ผ่อนคลาย ช่วยนอนหลับ
- **ซาติวา (Sativa)**: ให้พลังงาน กระปรี้กระเปร่า
- **ไฮบริด (Hybrid)**: สมดุลระหว่างทั้งสอง

## เทคโนโลยีที่ใช้

- React.js 18
- TailwindCSS
- Lucide React (icons)

## คำเตือน

เว็บไซต์นี้สร้างขึ้นเพื่อการศึกษาและสาธิตเทคนิคการพัฒนาเว็บไซต์เท่านั้น
การใช้กัญชาควรเป็นไปตามกฎหมายและข้อบังคับของประเทศนั้นๆ
